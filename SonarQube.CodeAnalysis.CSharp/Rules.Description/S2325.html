Class methods that don't access instance data can and should be static to prevent any misunderstanding about the contract of the method.

Noncompliant Code Example

public class Utilities
{
  public int Sum(int a, int b)  // Noncompliant
  {
    return a + b;
  }
}

Compliant Solution

public class Utilities
{
  public static int Sum(int a, int b)  // Compliant
  {
    return a + b;
  }
}

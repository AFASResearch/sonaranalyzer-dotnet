Using .Count() > 0 to test for emptiness works, but using .Any() makes the intent clearer, and the code more readable.

Noncompliant Code Example

private static bool HasContent(IEnumerable<string> strings)
{
  return strings.Count() > 0; // Noncompliant
}
private static bool HasContent2(<string> strings)
{
  return strings.Count() >= 1; // Noncompliant
}
private static bool IsEmpty(<string> strings)
{
  return strings.Count() == 0; // Noncompliant
}

Compliant Solution

private static bool HasContent(<string> strings)
{
  return strings.Any();
}

private static bool IsEmpty(<string> strings)
{
  return !strings.Any();
}

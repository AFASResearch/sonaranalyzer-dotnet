Using .Count() > 0 to test for emptiness works, but using .Any() makes the intent clearer, and the code more readable.

Noncompliant Code Example

private static bool HasContent(BlockSyntax block)
{
  return block.Statements.Count() > 0;
}

private static bool HasContent(BlockSyntax block)
{
  return block.Statements.Count() >= 1;
}
or 
private static bool IsEmpty(BlockSyntax block)
{
  return block.Statements.Count() == 0;
}

Compliant Solution

private static bool HasContent(BlockSyntax block)
{
  return block.Statements.Any();
}
or 
private static bool IsEmpty(BlockSyntax block)
{
  return !block.Statements.Any();
}
